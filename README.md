# 📊 Adidas Sales Dashboard | Power BI

An interactive **Power BI Sales Dashboard** built using the Adidas US Sales dataset to analyze sales performance, profitability, regional trends, retailer performance, and product-wise sales. The dashboard enables users to explore key business metrics through interactive visualizations and filters.

---

---

## 🎯 Project Overview

This project demonstrates how Power BI can transform raw sales data into meaningful business insights through interactive dashboards and visualizations.

The dashboard allows users to:

- Track overall sales performance
- Monitor operating profit and operating margin
- Analyze monthly sales trends
- Compare regional sales performance
- Explore state-wise sales distribution
- Identify top-performing products
- Compare retailer performance
- Filter data by region and invoice date

## 📊 Business Insights

- West region generated the highest sales.
- July recorded the highest monthly sales.
- West Gear was the top-performing retailer.
- Men's Street Footwear contributed the highest sales among products.
- The overall operating margin remained around **42%**.
